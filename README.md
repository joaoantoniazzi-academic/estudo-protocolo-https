# Análise dos Protocolos HTTP e HTTPS em Comunicação com Ecossistemas de Nuvem

Este repositório contém o artigo científico elaborado como objeto de pesquisa para o desenvolvimento do Projeto Integrador V e para a disciplina de Redes e Telecomunicações. O estudo aborda a evolução dos protocolos da camada de aplicação e transporte, seus mecanismos de segurança e o impacto prático dessa pilha tecnológica em ambientes *serverless*, utilizando o ecossistema Google Firebase/Firestore como caso de uso.

## Autores
- Guilherme Centenaro
-  João Henrique Antoniazzi
- Rafael César Michalczuk

## Resumo da Pesquisa

A consolidação da computação em nuvem e de sistemas distribuídos exigiu uma evolução drástica nos protocolos da internet. O modelo original TCP/IP trafegava dados em texto limpo (*cleartext*), gerando vulnerabilidades severas. Este trabalho analisa a transição para o ambiente seguro do HTTPS através de criptografia híbrida, a evolução do HTTP/1.1 até o HTTP/3 (QUIC), mecanismos de proteção ativa como o HSTS, e como essa infraestrutura se comporta em um ecossistema de nuvem real de alta performance (Firebase).
